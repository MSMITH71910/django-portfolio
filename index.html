Django CV Generator

Full-Stack CV / Resume Builder

A multi-section CV builder that lets users create, preview, and print a professional resume directly from the browser — no design software needed. Built end-to-end with Django 6, Bootstrap 5, and deployed to Render with PostgreSQL in production.

  • Django models for CV, Experience, Education, Skills, Projects & Certifications
  • Inline formsets — add or remove sections without page reloads
  • Live two-column preview with skill progress bars & photo upload
  • Print-ready A4 layout — browser print → Save as PDF
  • Full Django Admin panel for managing all CV data
  • WhiteNoise static files · Gunicorn · Render deployment via render.yaml
